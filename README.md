# RevenueOS AI (Next.js + Vercel)

This repository contains a deployable Next.js 15 starter that encodes a complete 2026 SaaS blueprint for an AI-powered marketing + sales growth operating system.

## Run locally

```bash
npm install
npm run dev
```

## Deploy to Vercel

1. Push repo to GitHub.
2. Import project in Vercel.
3. Set environment variables (AI keys, database URL, Stripe keys).
4. Deploy.

## Product blueprint covered in app UI

- App concept and target markets.
- Core modules: Audit, Fixer, Scripts, Viral Ads, Funnel Builder, Automations.
- User flow, AI architecture, monetization, and scale strategy.
- 30-day MVP execution plan.
